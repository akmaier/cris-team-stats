CRIS team stats — builder

Paste an FAU team-listing URL and get a sortable table of each member's CRIS stats. Use the Generate WordPress snippet button to get a copy-pasteable HTML block for your WordPress site.

Configuration

CRIS IDs are auto-discovered from each person's name. Rows that can't be matched keep an empty input where you can paste an ID manually; rows whose network request fails show a Retry search button.

Advanced

Only needed to override auto-discovery for specific rows (e.g. when two researchers share a surname).
